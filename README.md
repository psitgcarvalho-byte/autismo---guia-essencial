# Autismo: Guia Essencial - Landing Page

Página web estática para divulgação ou venda do ebook **Autismo: Guia Essencial**, de Thiago Carvalho.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie os arquivos deste pacote para o repositório.
3. Vá em **Settings > Pages**.
4. Em **Build and deployment**, selecione **Deploy from a branch**.
5. Escolha a branch `main` e a pasta `/root`.
6. Salve e aguarde o link ser gerado.

## Onde editar

- O arquivo principal é `index.html`.
- A imagem da capa fica em `assets/capa-autismo-guia-essencial.png`.
